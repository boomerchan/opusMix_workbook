These images were generated with a short and simple prompt. For example:
<lora:add_detail:0.4> (anime), 1girl, {prompt word}

The prompt words were tested on ~10 generations and were included here only if they consistently represented the subject matter.

Full-resolution previews with the original generation metadata can be found on this workbook's GitHub repo.
